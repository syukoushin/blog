Linux下安装nodejs

1、 下载notejs，附上下载地址
https://nodejs.org/en/download/

2、 解压
将下载的 node-v8.11.2-linux-x64.tar.xz 放到自己指定的目录下 我是放到了 /user/local/nodejs 下了，
在该目录下执行
tar -xf node-v8.11.2-linux-x64.tar.xz

3、配置环境变量
vim /etc/profile
在文件最后添加 export PATH=$PATH:/usr/local/nodejs/node-v8.11.2-linux-x64/bin
生效配置文件 source /etc/profile

至此nodejs安装结束
