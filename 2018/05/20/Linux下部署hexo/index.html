Linux下部署hexo

  1. 安装git，详见【Linux下安装git】

  2. 安装nodejs，详见【Linux下安装nodejs】

  3. 安装hexo 安装配置等详情可见官网 https://hexo.io/docs/index.html，
    以下仅给出简单步骤。

    ①.利用npm安装

    `npm install -g hexo-cli`

    ②.在目标文件夹初始化hexo,我选择的目标文件夹是/opt/www/blog

    `hexo init /opt/www/blog`

    ③.启动服务，在此指定了8008端口启动，默认4000，用的dubgg启动

    `hexo s -p 8008 --debug`

    ④.访问 ip:8008即可。
