Linux下安装git

1、 下载git，附上下载地址
https://github.com/git/git/releases/tag/v2.11.0

2、 解压
将下载的 git-2.11.0.tar.gz 放到自己指定的目录下 我是放到了 /user/local/git 下了，
在该目录下执行
tar -zxvf git-2.11.0.tar.gz

3、编译git
cd git-2.11.0
make prefix=/usr/local/git all
make prefix=/usr/local/git install
注：可通过git -–version 查看git版本验证是否安装完毕

4、配置环境变量
vim /etc/profile
在文件最后添加 export PATH=$PATH:/usr/local/git/bin
生效配置文件 source /etc/profile

至此git安装结束
